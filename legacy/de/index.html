Sicher

Starke Verschlüsselung

Bitcoin überprüft die Transaktionen mit einer auf einem Industriestandard basierenden Verschlüsselung, die von Regierungen und Militär verwendet wird.

Ganze Klassen von Fehler
sind nicht vorhanden.
Dan Kaminsky, Sicherheitsexperte

Offen

Vollständig dezentalisiert

Bitcoin ist Open Source Software. Niemand kann es kontrollieren. Die beliebteste Anwendung wird von einer Gemeinschaft von Open Source Entwickler gepflegt.

Fair

Minimale Gebühren

Die Benutzung des Bitcoin-Netzwerks ist bis auf eine freiwillige Gebühr kostenlos. Damit können Transaktionen beschleunigt werden.

Es ist der günstigste Weg
um Geld zu überweisen.
Max Keiser, Journalist & TV Moderator
