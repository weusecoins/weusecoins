Aidez-nous à traduire WeUseCoins

Le code et les instructions sont sur GitHub. Si vous voulez aider mais n'avez pas les aptitudes techniques nécessaires, vous pouvez nous envoyer un email à info@weusecoins.com (anglais seulement) afin que nous puissions vérifier si un développeur est disponible pour s'occuper de la partie technique. Veuillez prêter attention de respecter les lignes directrices dans le README et de fournir des traductions aussi exactes que possibles basées sur la version anglaise de WeUseCoins. Merci de votre aide!
